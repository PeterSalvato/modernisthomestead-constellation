# Modernist Homestead

Zero-willpower household systems for a neurodivergent family.

**Full case study:** [petersalvato.com/systems/modernist-homestead/](https://petersalvato.com/systems/modernist-homestead/)

## About

ADHD, autism, Celiac — the environment holds decisions so the people don't have to. Culinary modularity, sensory zoning, executive function scaffolding, and gamification. The system does the thinking when thinking is exactly what's constrained.
